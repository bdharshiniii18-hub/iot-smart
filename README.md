# IoT Based Smart Cane for Visually Impaired

## Description
This project focuses on developing an IoT-based smart cane to help visually impaired individuals detect obstacles and move safely.

## Technologies Used
- Arduino
- Ultrasonic Sensor
- Buzzer
- Embedded C
- IoT

## Features
- Obstacle detection
- Alert using buzzer
- Low-cost assistive device
